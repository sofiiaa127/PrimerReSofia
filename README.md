## Comandos git
### Traer los cambios de mi rama  remota
```git
git pull origin NOMBRE_RAMA
```
### añadir todos los cambios en stage

```git
git add .
```
### añadir todos un cambio específico en stage

```git
git add RUTA_DEL_ARCHIVO
```

### Hacer un commit
```git
git commit -m "QUE SE HIZO EN ESTOS CAMBIOS"
```

### Subir los cambios que ya tienen un commit en la rama actual
```git
git push origin NOMBRE-RAMA
```

### mirar todas las ramas
```
git branch -a
```

#### Si no aparecen las ramas nuevas que se han subido
```git
git fetch
```

### Navegar entre ramas; cambiar de rama
```git
git switch NOMBRE_RAMA
```
